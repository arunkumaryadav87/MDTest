# MDTest
Motion Design test

The test app has 5 screens

1) HomePage : This screen has 4 action buttons
  a) Add / Edit Constraint
  b) Add Item
  c) Show Package list
  d) Show Item list

2) Constraint Info: THis screen will open by tapping on Add/Edit constraint. It allows add or update package constraint

3) Add Item screen can be open using Add Item button from home screen. It allows to add new Items

4) Package List screen

5) Item List screen
